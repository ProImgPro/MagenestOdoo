Overview

This module helps to attend by using webcam for employees.

Installation

  • To install this module, you need also the hr module.
  • You need to install opencv , face_recognization library.

Hr Attendance Face Recognization

  • Quickly make roll-call with a snapshot
  • Set up FaceID information with user portrait image
  • Limit access to certain account to manage attendance

Screenshots

Go to Settings -> Users -> Choose users who you wanna have permission to use this attribute.

Now go to Employee -> Edit -> User will see "eye" icon appear.

Click "Take Snapshot" and "Save & Close" button. User can register face for employees. Algorithm will get employee's facial features in order to dectect face.

Now employees can check in / check out. Go Attendance -> Manager -> FaceID

Click Camera icon.

Click "Take Snapshoot" and "Check in" or "Check out" button.
